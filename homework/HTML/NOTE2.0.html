Color(顏色)

  • 顏色名稱表示

    color:darkblue

  • RGB表示

    color:rgb(0(R\紅色), 0(綠色), 139(藍色))

  • Hsl表示

    color:hsl(240(色相), 100%(飽和度), 27.3%(亮度))

  • 色碼表示

    color:#00008B

延伸有border-color、background-color......

Border(邊界)

border: X px(寬度) solid(樣式) darkblue(顏色)

樣式:solid(實線), dashed(虛線), double(雙線), dotted(點)......

可分別不同樣式 border-style:solid dotted dashed double; (自上順時鐘)

Padding(內距) & Margin(外距)

Padding為border內的距離(X px, X >= 0)

Margin為border外的距離(X px, X 可為負值, 會重疊)

padding or margin: X px

padding or margin-top: X px

padding or margin-right: X px

padding or margin-bottom: X px

padding or margin-left: X px

padding or margin: X px X px X px X px (自上順時鐘)
