簡志融的Css筆記

Css基礎概念

  1. 越裡面的Css style，優先度越高

    <div style="color:red">

      <h1 style="color:blue">Hello Css !!<h1>

    </div>

    則h1為藍色

  2. Css不能獨自存在，必搭配Html
  3. Css是網頁不可分割的一部分

Css寫法

  • 另外寫.css檔
  • 寫在<head>裡

    <head>

      <style>

        div{

          Css

        }

      </style>

    </head>

  • 寫在 Html tag 後面加 style="Css"

    <body style="Css">

    <div style="Css">

    <h1 style="Css">

    以此類推......

Class(類別)

<... class="類別名稱 類別名稱1">

每個html tag可有多個class標籤，以空格相隔

Id(唯一識別名稱)

<... id="識別名稱">

在整個網頁中，一個id只能給一個html tag，獨一無二

Css selector(選擇器)

可挑選有特定class、id或html tag套用Css

  • certain html tags

      <style>

        div{

          Css

        }

      </style>

  • all html tags

      <style>

        *{

          Css

        }

      </style>

  • class

      <style>

        .classname{

          Css

        }

      </style>

  • id

      <style>

        #id{

          Css

        }

      </style>

Color(顏色)

  • 顏色名稱表示

    color:darkblue

  • RGB表示

    color:rgb(0(R\紅色), 0(綠色), 139(藍色))

  • Hsl表示

    color:hsl(240(色相), 100%(飽和度), 27.3%(亮度))

  • 色碼表示

    color:#00008B

延伸有border-color、background-color......

Border(邊界)

border: X px(寬度) solid(樣式) darkblue(顏色)

樣式:solid(實線), dashed(虛線), double(雙線), dotted(點)......

可分別不同樣式 border-style:solid dotted dashed double; (自上順時鐘)

Padding(內距) & Margin(外距)

Padding為border內的距離(X px, X >= 0)

Margin為border外的距離(X px, X 可為負值, 會重疊)

  • padding or margin: X px
  • padding or margin-top: X px
  • padding or margin-right: X px
  • padding or margin-bottom: X px
  • padding or margin-left: X px
  • padding or margin: X px X px X px X px (自上順時鐘)

Display(顯示)

display可改變容器顯示性質

如<p style="display:inline">便會使得p從原本占一行，變成有多少占多少

如<span style="display:block">便會使得span從原本有多少占多少，變成占一行

Float(懸浮)

float猶如word的版功能，可使文字排在圖片左方或右方

<img style="float: right;" src="圖片連結/位置/名稱"/>
